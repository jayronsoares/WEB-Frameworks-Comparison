What if you could see the same project developed with the most famous WEB frameworks?

The Project

I will try to develop a Task List application on few of the most famous WEB framworks of PHP, Python and Ruby.. and this list of languages will grow for sure... Maybe node.js?

Why

This idea came after reading the article PHP: a fractal of bad design, not because the "PHP is bad" part, but more because I wanted to learn about how others build the WEB that we know.

I want to study and practice other programming languages​​, that's why I decided to create this project.

I was also inspired by the Coding Dojo way of life.

Who am I

My name is Thiago Belem, I'm a web developer and I work mainly with PHP... Yeah, I love CakePHP.

I'm also the creator and teacher of Assando Sites, an online CakePHP course.
